# Reproducibility Package

Minimal, standalone repository to reproduce the main `paper_6` AVAMET-IMERG experiment and manuscript figures.

This repository keeps only the scripts needed for the principal pipeline:

- station subset and IMERG alignment
- IMERG CV subset extraction
- AVAMET QC and event catalog generation
- AVAMET-IMERG event matching and window-based verification
- amplitude-displacement analysis
- outlier filtering, stratification, and DANA case summaries
- main manuscript figures

It intentionally excludes exploratory, sensitivity, and supplementary scripts that are not required for the core rerun.

## Data Access

The input datasets are not redistributed in this repository.

- `data/avamet_all.parquet` or `data/avamet_all_2019_2025_0930.parquet` is expected as the AVAMET input parquet.
- `data/imerg/` should contain raw GPM IMERG Final V07 half-hourly HDF5 files.
- `data/imerg_cv/` is a derived subset written by the extraction step and is also ignored by git.

If you already have the extracted IMERG subset, you can skip the extraction step with `--skip-extraction`.

## Expected Layout

```text
paper_6_minimal/
  data/
    avamet_all.parquet
    imerg/
      YYYY/...*.HDF5
    imerg_cv/
      YYYY/...*.cv.npz
  results/
    figures/
  scripts/
  run_pipeline.py
  requirements.txt
```

Notes:

- `data/avamet_all.parquet` is the default AVAMET input. You can override it in `run_pipeline.py`.
- `data/imerg/` should contain raw IMERG Final V07 half-hourly HDF5 files.
- `data/imerg_cv/` is generated by step `02_extract_imerg_cv_subset.py`. If it already exists, run with `--skip-extraction`.

## Install

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

Tested with Python `3.13.2`.

## Run

Full pipeline from raw AVAMET + raw IMERG:

```bash
python run_pipeline.py --avamet-input data/avamet_all_2019_2025_0930.parquet
```

If `data/imerg_cv/` is already available:

```bash
python run_pipeline.py --avamet-input data/avamet_all_2019_2025_0930.parquet --skip-extraction
```

## Reproducibility Notes

- Scripts `00` and `01` download Natural Earth admin boundaries at runtime, so internet access is required unless you vendor that boundary data separately.
- No input data are included in this repository.
- This minimal repo starts from raw AVAMET parquet plus raw IMERG HDF5, or from an already extracted `data/imerg_cv/` subset.
